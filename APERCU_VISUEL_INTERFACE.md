## Effets Visuels

### État Normal
```
┌──────────────────┐
│   📖 Livre       │  ← Border radius: 20px
│                  │  ← Shadow: douce
│   Auteur         │  ← Background: blanc
└──────────────────┘
```

### État Hover
```
    ┌──────────────────┐
    │   📖 Livre       │  ← Élévation: -8px
    │                  │  ← Shadow: grande + violet
    │   Auteur         │  ← Scale: 1.02
    └──────────────────┘
        ↑ Transform: translateY(-8px)
```

### Loading State
```
┌──────────────────┐
│   ⏳ Livre       │  ← Opacity: 60%
│                  │  ← Pointer: not-allowed
│   ...            │  ← Spinner animé
└──────────────────┘
```
