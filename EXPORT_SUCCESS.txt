🚀 INSTALLATION RAPIDE
═══════════════════════════════════════════════════════════════

MÉTHODE 1: WordPress Admin (5 minutes)
───────────────────────────────────────

1. Extensions → Ajouter → Téléverser une extension
2. Installer dans l'ordre:
   → cpfa-core-manager.zip
   → cpfa-forms-registrations.zip
   → cpfa-pdf-generator.zip
3. Activer chaque plugin après installation

⚠️  ORDRE CRITIQUE: Core Manager en premier !

MÉTHODE 2: FTP/SFTP (10 minutes)
─────────────────────────────────

1. Extraire cpfa-production-20251012-220119.tar.gz
2. Uploader les 3 dossiers cpfa-* vers /wp-content/plugins/
3. Activer les plugins via WordPress Admin

💡 ASTUCES
═══════════════════════════════════════════════════════════════

Problème: Fichier trop gros (cpfa-pdf-generator.zip = 59 MB)
Solution 1: Augmenter upload_max_filesize à 64M
Solution 2: Installer via FTP (pas de limite)

Problème: Dashboard affiche 0 partout
Solution: Attendre 1-2 minutes (cache), puis rafraîchir

Problème: Emails non reçus
Solution: Installer WP Mail SMTP
